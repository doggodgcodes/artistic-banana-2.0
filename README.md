# artistic-banana-2.0
Artistic Banana by minechicken98 on github, but better! (credits to minechicken98 for the original source code)

Artistic Banana is an HTML canvas drawing game by **minechicken98.**

The original code is at [minechicken98/funny](https://github.com/minechicken98/funny)

Most of the source coded was written by the computer but I wrote some code and tweaked things

# Improvements from the original
- Colors
  - Organized default colors
  - Hex colors support
  - Rainbow order colors and black and white for default colors
  - Shifted magenta to purple to better match the rainbow
- Brushes
  - New brush size (1px)
- Item groups
  - Updated text to be blue
  - Updated text to be lowercase instead of uppercase
  - Color picker renamed to Advanced Colors
  - Quick Colors renamed to Default Colors
  - Actions
    - Actions renamed to canvas utilities
    - Clear renamed to Clear Canvas
    - Save renamed to Save as PNG
  - Tip updated to include tap to signify touchscreen support
  - Create your masterpiece renamed to I'm a goofy goober
  - Hex code list with color descriptions
  - Small CSS updates and   more padding for the body
